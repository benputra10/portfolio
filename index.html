Skills

HTML5

have a strong command of HTML5, the core markup language for modern web development. With HTML5, I can build well-structured, semantic, and SEO-friendly web pages. I am proficient in using new elements such as <article>, <section> and <figure>, ensuring better accessibility and search engine optimization. and optimized website performance.

CSS3

I specialize in CSS3 to create visually appealing and responsive web designs. My expertise includes using Flexbox, Grid, and media queries to craft layouts that adapt seamlessly across different devices. Additionally, I utilize animations, transitions, and CSS Variables to enhance user experience and improve the overall aesthetics of a website.

JAVASCRIPT
